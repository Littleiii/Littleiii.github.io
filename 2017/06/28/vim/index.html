注^(*￣(oo)￣)^:

在正则表达式中，^表示匹配字符串的开始位置，$表示匹配字符串的结束位置。命令前面加数字表示重复的次数，加字母表示使用的缓冲区名称。使用英文句号”.”可以重复上一个命令。
