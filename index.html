git常用命令总结

创建操作

  • mkdir (name) 创建目录
  • cd (name) 进入到该目录
  • pwd 查看当前目录路径
  • git init 将目录变成Git管理仓库
  • git add name 将库中文件添加到暂存区
  • git add -a 将库中所有文件添加到暂存区
  • git commit -am “提示信息” 将文件提交到仓库

什么是代理模式?

App开发规范

App开发规范

一.项目结构

随着MVP设计模式的流行,就默认采用下面两种项目结构.

  1. 适合小项目
    • app——Application Activity Fragment
    • Presenter等的顶级父类
    • config——API,常量表等
    • model——数据层
    • entities——数据模型
    • presenter——MVP的P
    • view——MVP的V
    • utils——工具类集合
    • widget——各个可复用View集合
    • ……
  2. 适合大项目
    • app
    • config
    • model
      • entities
    • module——将界面层以功能模块分配包
      • launch
      • main
      • account
      • news
      • music
      • ……….
    • ……
